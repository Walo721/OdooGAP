This module introduces a new dashboard for employee performance metrics, including:

* Sales per employee
* CRM opportunities per employee
* Expenses per employee

It adds a new menu in the Dashboard app to display these charts.

Usage
=====

To use this module, you need to:

1. Go to the Dashboards app.
2. Click on the "Employee Performance" menu.
3. View the charts.
